# 凡人残图

一个面向《凡人修仙传》爱好者的追番情报与内容导航项目。

第一版以“每周残图”为核心，把正片更新时间、剧情坐标、优质解析、二创入口、订阅提醒与下集预测组织成一张可以探索的古卷地图。

## 当前版本

- 原创古卷地图首屏与四类可点击内容节点
- 自动计算下一次周六 11:00 更新倒计时
- 本周解析与二创筛选
- 本地储物袋收藏
- 本地传音提醒状态
- 无金钱、不可交易的下集预测与签到积分演示
- 公开链接采集器的产品状态展示

当前内容数据为产品演示。后续将接入合规的公开信息采集、去重与原址导航服务，不保存或重新分发正片和创作者正文。

## 本地运行

需要 Node.js 22 或更高版本。

```bash
npm run dev
```

访问终端中显示的本地地址即可。

## 构建

```bash
npm run build
```
